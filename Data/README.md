## Data Source

The dataset used in this project comes from Kaggle:

**Plant Communication Dataset — Classification**

https://www.kaggle.com/datasets/efeyldz/plant-communication-dataset-classification/data

---

## Description

This dataset contains measurements related to plant communication signals and environmental features.  
It is designed for supervised classification tasks, where the goal is to predict plant interaction or response categories based on observed attributes.

The dataset includes:

- Multiple numerical sensor features
- Signal-related measurements
- Target class labels for classification
